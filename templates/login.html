Acerca de Nosotros

Centro Minero SENA

Regional Boyacá

El Centro Minero del SENA es una institución líder en la formación profesional integral en el sector minero y energético de Colombia.

Sistema de Gestión de Laboratorios

Plataforma moderna para la administración eficiente de recursos educativos, equipos y espacios de laboratorio.

  • Control de acceso con reconocimiento facial
  • Gestión inteligente de equipos e inventario
  • Reservas automatizadas
  • Reportes en tiempo real

Ayuda
¿Cómo iniciar sesión?
  1. Ingresa tu ID de usuario (Ej: INST001, APRE001)
  2. Ingresa tu contraseña
  3. Haz clic en "Iniciar Sesión"
Reconocimiento Facial

Puedes usar el botón de cámara para autenticarte mediante reconocimiento facial. Asegúrate de tener buena iluminación.

¿Olvidaste tu contraseña?

Haz clic en "¿Olvidaste tu contraseña?" y sigue las instrucciones para recuperarla por correo electrónico.

Para más ayuda, contacta al administrador del sistema.
Política de Privacidad

Última actualización: Octubre 2025

Información que Recopilamos
  • Datos de identificación personal
  • Datos biométricos (reconocimiento facial)
  • Registros de uso del sistema
Uso de la Información

La información se utiliza exclusivamente para la gestión de laboratorios y control de acceso.

Protección de Datos

Implementamos medidas de seguridad técnicas y organizativas para proteger tus datos personales.

Términos de Uso

Última actualización: Octubre 2025

Aceptación de Términos

Al acceder al sistema, aceptas estar sujeto a estos términos y condiciones.

Uso Autorizado

El sistema está destinado exclusivamente para aprendices, instructores y personal autorizado del SENA.

Responsabilidades
  • Mantener la confidencialidad de tus credenciales
  • Usar el sistema solo para fines educativos
  • Reportar cualquier problema de seguridad
Accesibilidad

El sistema incluye opciones de accesibilidad para garantizar que todos los usuarios puedan utilizarlo:

  • Alto Contraste: Mejora la visibilidad del texto
  • Tamaño de Texto: Ajustable de 80% a 200%
  • Navegación por Teclado: Compatible con lectores de pantalla
  • Comandos de Voz: Control por voz disponible

Estas opciones estarán disponibles una vez inicies sesión.
