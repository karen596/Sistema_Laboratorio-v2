  2. REDISEÑO COMPLETO DEL DASHBOARD
═══════════════════════════════════════════════════════════════════

🎨 NUEVA PALETA DE COLORES
   -------------------------
   Verde Emerald Focus:
   - Primary: #2ECC71 (Emerald Green vibrante)
   - Dark: #1E392F (Jungle Green para sidebar)
   - Light: #00D27F (Energético para CTAs)
   - Subtle: #F1FBF7 (Mint Cream para hover)

   Neutrales Clean:
   - Background: #F9FAFB (Off-white)
   - Cards: #FFFFFF (Blanco puro)
   - Text: #2D3748 (Charcoal)
   - Secondary: #6B7280 (Gris)

   Dark Mode:
   - Background: #1A202C (Near-black)
   - Cards: #2D3748 (Dark gray)
   - Accent: #2ECC71 (Alto contraste)

📐 COMPONENTES MODERNIZADOS
   -------------------------
   
   KPI Cards:
   - Números grandes (52px) en verde emerald
   - Iconos en fondos mint cream
   - Hover con elevación y sombra
   - Border radius 12px

   Action Cards:
   - Diseño minimalista
   - Barra superior verde animada
   - Flecha → que aparece en hover
   - Toda la card clickeable
   - Transiciones cubic-bezier suaves

   Sidebar:
   - Color Jungle Green (#1E392F)
   - Links activos con fondo Mint Cream
   - Iconos outline unificados
   - Hover states con desplazamiento

   Charts:
   - Gradiente verde (Jungle → Emerald → Energetic)
   - Tooltips modernos
   - Grid lines sutiles

   Activity Feed:
   - Skeleton loaders animados
   - Items con borde izquierdo verde
   - Hover con desplazamiento

🔤 TIPOGRAFÍA
   -----------
   Font: Inter (moderna y legible)
   - Page Title: 32px, Bold
   - KPI Values: 52px, Bold, Emerald
   - Section Titles: 20px, SemiBold
   - Body: 15px, Regular
   - Labels: 14px, Medium, Uppercase

📏 ESPACIADO & LAYOUT
   ------------------
   - Whitespace generoso (2.5rem entre secciones)
   - Max-width: 1400px
   - Border-radius: 12px consistente
   - Sombras sutiles (0.02 - 0.15 opacity)
   - Grid responsive (4→2→1 columnas)

🌙 DARK MODE
   ----------
   - Soporte completo implementado
   - Activación desde modal de accesibilidad
   - Variables CSS específicas
   - Alto contraste con Emerald Green

  6. CÓMO PROBAR LOS CAMBIOS
═══════════════════════════════════════════════════════════════════

1. INICIAR SERVIDOR
   python web_app.py

2. ACCEDER AL DASHBOARD
   http://localhost:5000/dashboard
   - Ver nuevo diseño
   - Probar hover effects
   - Verificar responsive

3. PROBAR DARK MODE
   - Click en botón de accesibilidad (navbar)
   - Activar "Modo Oscuro"
   - Guardar configuración

4. VER SISTEMA DE DISEÑO
   http://localhost:5000/design-system
   (Solo administradores nivel 4)

5. PROBAR ELIMINAR BACKUP
   http://localhost:5000/backup
   - Click en botón de eliminar
   - Confirmar en modal
   - Verificar mensaje de éxito

  9. NOTAS TÉCNICAS
═══════════════════════════════════════════════════════════════════

🔧 Variables CSS
   Todas las variables están en :root en main.css
   Usar var(--nombre-variable) para consistencia

🔧 Clases de Utilidad
   - .kpi-card, .action-card, .chart-card
   - .section-spacing, .section-header
   - .skeleton para loaders
   - body.modo-oscuro para dark mode

🔧 Breakpoints
   - Desktop: >992px
   - Tablet: 768-991px
   - Mobile: <768px

🔧 Iconos
   Bootstrap Icons - Preferir outline style
   Tamaño: 1.5rem para KPI, 1.75rem para actions
